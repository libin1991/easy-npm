## transition
> 参数

|    属性   |  类型  | 默认 | 必传 |                                     描述                                     |
|:---------:|:------:|:----:|:----:|:----------------------------------------------------------------------------:|
| animation | string | 空   | N    | 取值范围：fade zoom slideDown slideLeft slideRight slideUp flip rotate door  |

> 使用
```
<Transition animation={animation}>
    <youComponent/>
</Transition>
```
